# Experiment 5: Implementation of Linear and Logistic Regression

## Description
This repository contains Experiment 5 of the Machine Learning Lab, which focuses on implementing Linear Regression and Logistic Regression using Python.

Linear Regression is a supervised learning algorithm used for predicting continuous values, while Logistic Regression is used for classification problems. In this experiment, both models are implemented to understand their working, training process, and performance evaluation.

## Features / Operations Covered
- Importing required libraries (NumPy, Pandas, Scikit-learn)
- Splitting dataset into training and testing sets
- Implementing Linear Regression model
- Implementing Logistic Regression model
- Model training and prediction
- Evaluating model performance
- Understanding difference between regression and classification

## Learning Objective
To understand and implement fundamental supervised learning algorithms (Linear and Logistic Regression) and analyze their performance on real datasets.

## Applications in Machine Learning
- Linear Regression: Predicting continuous values (e.g., price prediction)
- Logistic Regression: Classification tasks (e.g., spam detection, survival prediction)

## Conclusion
This experiment provides a strong foundation in supervised learning techniques. Understanding Linear and Logistic Regression is essential for building and evaluating machine learning models.
